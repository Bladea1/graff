# GraffZone (Android, offline-first)

Учебное приложение «соцсеть про граффити» без сервера: все данные хранятся **локально** (Room DB).
Сборка делается через **GitHub Actions** и/или локально через Android Studio.

## Что внутри (MVP)
- Лента постов (картинка/видео, теги, лайки, комментарии)
- Explore (поиск по тегам/тексту)
- Карта (OpenStreetMap через osmdroid) с метками постов с локацией
- Ивенты (локально)
- Чаты (локально)
- Профиль (локально)
- Тёмная тема

## Быстрый запуск локально (Android Studio)
1) Установи Android Studio.
2) Открой папку проекта.
3) Дай Gradle скачать зависимости (первый Sync).
4) Run ▶️

> В проекте **gradle-wrapper.jar не хранится**, но `./gradlew` сам скачает его при первом запуске
> (это сделано, потому что jar — исполняемый бинарник).

## Сборка APK через GitHub Actions
1) Создай репозиторий на GitHub и залей сюда весь код.
2) Открой вкладку **Actions** → запусти workflow **Android CI**.
3) Скачай Artifact `GraffZone-debug-apk` → внутри будет `app-debug.apk`.

## Примечания
- Карта использует онлайн-тайлы OpenStreetMap (сервер проекта не нужен).
- Никаких реальных аккаунтов/бэкенда — всё локально.
